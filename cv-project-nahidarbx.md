# NahidArbX: Real-Time Value-Bet Detection Platform

**Organization:** Personal Project / Self-Directed

**Date:** June 2026 - Present

**Project URL:** Add your GitHub, portfolio, or live demo URL here

## Description

- Built **NahidArbX**, a **real-time sports betting analytics platform** that compares soft-book odds against sharp market prices to detect positive expected value opportunities.
- Developed a dual-process architecture using **Next.js**, **TypeScript**, **PostgreSQL**, and **Drizzle ORM** for the frontend, engine, database layer, and background workers.
- Implemented **provider integrations**, odds normalization, event matching, value-bet detection, deterministic settlement, and Telegram notifications.
- Built operational dashboards for value bets, bet history, matcher review, ML pipeline monitoring, provider health, and system logs.
- Integrated **Google Cloud** infrastructure including **Cloud SQL** and **Cloud Run Jobs** for managed database access and ML training workflows.
- Designed an **ML pipeline** using **LightGBM** to support model training, learning observability, and betting performance analysis.
