# Bet Placement UI Revamp Plan

## Current State

**Location:** `components/spreadsheet/ValueBetDetailsModal.tsx` (lines 1109-1564)

**Current Layout:**

- Dark panel with emerald accents
- Two-column grid: stats matrix (left) + interaction area (right)
- Stats: Book, Odds, Edge, Min, Max, Balance
- Stake input with currency suffix
- "Ignore EV" checkbox for negative EV
- Confirm Bet button
- Result display area

---

## Revamp Goals (Clean Minimal)

1. **Simplify visual design** - flat, clean aesthetic with minimal decoration
2. **Improve information hierarchy** - clear horizontal layout with dividers
3. **Streamlined UX** - fewer colors, more whitespace, focused interactions
4. **Better mobile responsiveness** - stacked layout for mobile

---

### Key Changes (Clean Minimal Style)

1. **Refined Stats Layout**
   - Horizontal layout with clear separation
   - Monospace numbers for alignment
   - Subtle dividers instead of borders
   - Generous padding and spacing

2. **Enhanced Stake Input**
   - Clean, borderless input with bottom underline
   - Quick preset buttons (Kelly, Min, Half, Max)
   - Subtle limit indicators below input

3. **Better Result Display**
   - Simple success/error banners
   - Clean typography with clear hierarchy
   - Minimal animations (fade only)

4. **Improved Validation**
   - Underline color changes for feedback
   - Simple inline text for errors
   - No tooltips - keep it on-page

5. **Reduced Visual Noise**
   - Remove gradient accents
   - Reduce number of colors
   - Flat design with subtle shadows
   - Focus on content over decoration
